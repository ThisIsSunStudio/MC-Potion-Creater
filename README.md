# MC-Potion-Creater

这是一个在**1.20.5**之后，用[give指令](https://zh.minecraft.wiki/w/Give)和[物品堆叠格式](https://zh.minecraft.wiki/w/%E7%89%A9%E5%93%81%E5%A0%86%E5%8F%A0%E7%BB%84%E4%BB%B6)的一个药水生成器。拥有多药水、颜色、其他自定义属性等功能。可以设置单个药水的属性，如是否为信标效果、是否显示粒子、是否显示图标([详情](https://zh.minecraft.wiki/w/%E7%89%A9%E5%93%81%E5%A0%86%E5%8F%A0%E7%BB%84%E4%BB%B6#potion_contents))的功能。
